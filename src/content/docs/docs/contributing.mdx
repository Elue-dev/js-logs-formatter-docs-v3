Here is some description about contributing to the project. Follow the steps below to set up the project locally.

<Steps>
  <ol>
    <li>
      <h3>Step 1: Clone the Repository</h3>
      <p>
        First, clone the repository to your local machine by running the following command:
      </p>
      ```sh
      git clone https://github.com/Elue-dev/js-logs-formatter.git
      ```
    </li>
    <li>
      <h3>Step 2: Navigate to the Project Directory</h3>
      ```sh
      cd js-logs-formatter
      ```
    </li>
    <li>
      <h3>Step 3: Install Dependencies</h3>

      <Tabs>
        <TabItem label="pnpm">
          ```sh
          pnpm install
          ```
        </TabItem>
        <TabItem label="npm">
          ```sh
          npm install
          ```
        </TabItem>
        <TabItem label="yarn">
          ```sh
          yarn install
          ```
        </TabItem>
        <TabItem label="bun">
          ```sh
          bun install
          ```
        </TabItem>
      </Tabs>
    
    </li>

    <li>
      <h3>Step 4: Make Your Code Changes</h3>
      <p>
        Now that you have the project set up, you can start making your code changes or implementing new features. Here are some tips to keep in mind:
      </p>
      <ul>
        <li><strong>Follow Coding Standards:</strong> Ensure your code adheres to the project's coding conventions.</li>
        <li><strong>Test Your Changes:</strong> Write tests to cover your changes, ensuring everything works as intended.</li>
        <li><strong>Commit Often:</strong> Make small, frequent commits with clear messages to document your progress.</li>
      </ul>
    </li>

    <li>
      <h3>Step 5: Open a Pull Request</h3>
      <p>
        Once you've made your changes and are satisfied with your work, itâ€™s time to submit a Pull Request (PR):
      </p>
      <ol>
        <li>
          <strong>Push Your Changes:</strong> Push your branch to your forked repository:
          ```sh
          git push origin your-branch-name
          ```
        </li>
        <li>
          <strong>Open a Pull Request:</strong> Go to the original repository on GitHub and click on the "Pull Requests" tab. Click "New Pull Request," select your branch, and provide a clear description of your changes.
        </li>
        <li>
          <strong>Request Review:</strong> After creating the PR, you can request reviews from the maintainers to get feedback on your work and merge the PR.
        </li>
      </ol>
    </li>
  </ol>
</Steps>
